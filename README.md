This is a simple Rock-Paper-Scissors game built with JavaScript. The game plays 5 rounds between a human player and the computer, keeping score and declaring a winner at the end.

## Features
- Random computer choice
- User input via prompt
- Score tracking
- 5-round match logic
- Console-based gameplay

## How to Play
1. Open `index.html` in a browser.
2. Follow the prompt to enter your choice: `rock`, `paper`, or `scissors`.
3. The computer will randomly choose its move.
4. After 5 rounds, the game will declare the winner.
